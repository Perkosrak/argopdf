Description

ArgoPDF is a pdf report generator tool for ArgoUML that is developed by Dzmitry Churbanau during the Google Summer of Code 2007 program.

The Mission

The aim of this project is to produce a module for ArgoUML that can generate pdf reports of data contained in an ArgoUML project. The module will use a free Java library iText. The reason for doing this is that it will make much easier to maintain the documentation of any data, contained in ArgoUML. The user just need to select what he wants to see in the report and the report will be generated on the fly.

In this document i will try to define components of ArgoUML, which can be included in the report.

In general, project in ArgoUML consists of next components:

Model: Package, Diagram
Package: Package, Diagram
Diagram: Class Diagram, UseCase Diagram, Sequence Diagram, Collaboration Diagram, Statechart Diagram, Activity Diagram, Deployment Diagram
Class Diagram: Package, Class, Interface, Enumeration
